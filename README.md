## ✨ What is UseTheseTools?
A collection of **33+ free online tools** built for speed, privacy, and ease-of-use. Every tool runs 100% in your browser — your data never touches a server.

## 🔧 Featured Tool Categories
*   **🌙 Astrology & Destiny**: Moon phase calculator, love compatibility, past life origin, daily horoscopes, tarot.
*   **🛠️ Developer Utilities**: JSON formatter & validator, Base64 encoder/decoder, JWT debugger, URL encoder/decoder.
*   **⚡ Productivity**: AI text sanitizer, reading level checker, password strength meter, focus timer with ambient sounds.
*   **🎨 Design & Fun**: Social media safe zone checker, birthday card generator, pet name generator, QR code generator.

## 💡 Why These Tools?
*   **🧠 100% Client‑Side**: Everything works offline after loading. No backend, no server costs.
*   **🔒 Privacy First**: No data collection. No cookies. No tracking scripts (except optional analytics).
*   **📱 Fully Responsive**: Works perfectly on mobile, tablet, and desktop.
*   **♿ Accessible**: Semantic HTML, ARIA labels, and keyboard navigation ready.

## 🛠️ Tech Stack
*   `HTML5` • `CSS3` • `Vanilla JavaScript` • `Web Audio API` • `Local Storage` • `QRCode.js` Library
