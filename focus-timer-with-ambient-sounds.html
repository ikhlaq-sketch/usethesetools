How it works

Choose a timer mode (Pomodoro, short break, long break). Start the countdown – you'll hear a gentle beep when time is up. Enable ambient sounds (rain, café, brown noise) to create a distraction‑free environment. All sounds are generated locally using the Web Audio API – no external files, completely private.

🔊 The rain sound is synthesized filtered noise; café is a low‑fi background murmur; brown noise is deep and soothing. Adjust volume sliders independently.

FAQs

Are the sounds recorded?
No – they are generated in real‑time using the Web Audio API. Nothing is downloaded, and your privacy is absolute.
Can I use this offline?
Yes, after first load the timer and sounds work without internet.
Does it work on mobile?
Yes – fully responsive. Some mobile browsers require user interaction to start audio; tap the 'Play' buttons to activate.
